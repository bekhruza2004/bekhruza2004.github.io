Movie Webpage

Graveyard Pact

Context

The project prompt was to create a fictional movie webpage using HTML and CSS. The webpage is themed around a horror movie titled "Graveyard Pact." The requirements specified index.html file, a style.css file, and local images, demonstrating learned skills concerning web design and layout.

Content

1. Overview of the Project: The webpage showcases the movie "Graveyard Pact," telling the story of a group of friends who uncover and confront dark supernatural forces in an eerie graveyard. 2. Files Created: index.html: Contains the main structural markup for the movie's presentation. style.css: Manages visual styles, including typography, colors, and layout. plan.md: Outlines project requirements, partner responsibilities, and stylistic choices. 3. Key HTML Structure: The webpage begins with a main title (h1), promoting the film's name. Several sections (div elements) introduce various plot points and themes, organized under headings (h2) and supporting paragraphs (p). It includes form elements where users can input their favorite horror shows, enhancing interactivity. 4. CSS Styling: CSS is used to enhance the visual appeal of the webpage: Custom fonts from Google Fonts add unique styles. Styling classes for headings, backgrounds, and images create a cohesive and atmospheric design. The dark background class provides a design, while specific styles for images and headings ensure they capture attention without overwhelming the viewer. 5. Local Image Usage: Local images for the graveyard and a scary statue enhance the thematic elements of the site, ensuring it aligns with the horror genre.

Reflection

Takeaways: Creating this movie webpage provided a practical application of HTML and CSS concepts learned in class. The project reinforced: Collaboration: Coordinating with a partner involved dividing tasks based on each person's strengths and preferences, ensuring an efficient workflow. Design Choices: Selecting appropriate colors and fonts was crucial in establishing the horror theme. The dark background and specific font styles contributed to creating an eerie atmosphere, which is essential in engaging the audience with a horror narrative. User Interaction: Including forms (input and button) added a layer of interactivity that encourages viewers to engage more with the webpage. Importance of Planning: Completing the plan.md helped lay out essential components and set clear responsibilities, making the production process smoother and more organized. Overall, the project provided a comprehensive introduction to web development, underscoring the significance of both structure and design in creating a functional and inviting website.
